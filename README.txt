OpenMP unter MacOS:
- Laden Sie mithilfe von Homebrew OpenMP: brew install libomp
- Damit der Compiler diese Bibliothek findet muss folgendes im ~\.zshrc eingetragen werden: export OpenMP_ROOT=$(brew --prefix)/opt/libomp
